This project started off as a way to learn and practice my cyber skills.

It began as a project to set up a self-hosted website from my HomeLab.

As it was on a budget, I looked at the cheapest ways to host this website.

As it turns out, hosting on the darkweb is free as I don't need to pay a registrar for a domain name.

I realised it was a pretty cool thing to be self-hosting my website on the darkweb so I decided to stick with this solution.

There will be many people who for whatever cannot or don't want to access my darkweb website.

Also, the darkweb is slow!

As such I've decided that I will host a clone of this web-page on GitHub Pages.

Once the GitHub Pages clone is up and running, I will post the clear web link here.
